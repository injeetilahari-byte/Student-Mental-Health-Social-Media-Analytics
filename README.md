# 📊 Student Mental Health & Social Media Analytics

## Overview

This Power BI dashboard analyzes the relationship between social media usage, lifestyle habits, and mental health outcomes among students worldwide.

The project explores how factors such as social media usage, sleep duration, exercise frequency, and demographic characteristics impact student wellbeing.

---

## Dataset Information

- 👨‍🎓 Total Students: 5,000
- 🌍 Countries Covered: 111
- 📑 Features: 13 Columns

### Key Variables

- Age
- Gender
- Country
- Social Media Usage Hours
- Mental Health Score
- Stress Level
- Anxiety Level
- Sleep Hours
- Exercise Frequency
- Platform Usage

---

## Project Objectives

- Analyze student social media usage patterns
- Explore factors affecting mental health
- Compare trends across different countries
- Identify lifestyle habits linked to wellbeing
- Present insights through interactive dashboards

---

## Dashboard Pages

### 1️⃣ Executive Overview
Provides high-level KPIs and summary insights.

### 2️⃣ Social Media Analytics
Analyzes platform usage trends and engagement patterns.

### 3️⃣ Mental Health Analytics
Examines stress levels, anxiety indicators, and mental health scores.

### 4️⃣ Lifestyle Analytics
Explores relationships between sleep, exercise, and wellbeing.

### 5️⃣ Country Analysis
Compares student wellbeing metrics across countries.

### 6️⃣ Country Detail Analysis
Provides detailed drill-through insights for individual countries.

---

## Key Insights

- Higher social media usage is associated with lower average mental health scores.
- Students with better sleep habits tend to report improved wellbeing.
- Regular exercise is linked to reduced stress levels.
- Significant variations exist across different countries and demographics.
